# PRODIGY_SD_02 - Number Guessing Game

## About the Project

This project is a simple Number Guessing Game developed in Python as part of my Prodigy InfoTech Software Development Internship.

The game generates a random number between 1 and 100, and the user has to guess the correct number. After each guess, the program provides a hint indicating whether the guessed number is too high or too low. The game continues until the correct number is guessed and displays the total number of attempts taken.

## Features

* Generates a random number for each game
* Accepts user input from the keyboard
* Provides hints to help the user guess correctly
* Counts the number of attempts
* Easy-to-use command-line interface

## Technologies Used

* Python 3
* Random Module

## How to Run

1. Download or clone this repository.
2. Open the project folder in VS Code.
3. Run the following command in the terminal:

python guessing_game.py


4. Start guessing the number and enjoy the game!
